Подробное описание

Класс для работы с длинными целыми числами.

Класс предоставляет методы для создания длинных целых чисел из списка инициализации, вектора или строки, а также методы для получения знака числа, доступа к цифрам числа, получения вектора цифр, представления числа в виде строки, получения количества цифр. Число хранится в виде пары (знак, вектор цифр).
