Подробное описание

Класс для работы с длинными натуральными числами с нулем.

Класс предоставляет методы для создания длинных натуральных чисел из списка инициализации, вектора или строки, а также методы для получения различных представлений числа. Число хранится в виде вектора цифр, где 0-й элемент - старший разряд.
